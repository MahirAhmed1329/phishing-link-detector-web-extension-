🏗️ Architecture Changes

  • utils.jsNEW — Shared scoring + URL normalisation injected before the content script. Single source of truth for calcRiskScore() and scoreToVerdict().
  • New scoring formulaUPDATED — (malicious + suspicious×0.5) / total × 100. Categories: 0–20 Safe, 21–60 Suspicious, 61–100 Malicious.
  • Canvas donut chartNEW — Pure Canvas API, zero dependencies. Animated ease-out draw. Shows real engine breakdown (mal / susp / safe / undetected).
  • Icons (16/32/48/128px)NEW — Auto-generated from your uploaded logo PNG. Registered in manifest.json for both icons and action.
  • Context menuNEW — Right-click any link → "Scan with PhisShield" → desktop notification with verdict.
  • Session statsNEW — Popup shows live scans / cache hits / error counts from the background service worker.
  • Rate-limit handlingUPDATED — 429 responses parsed for Retry-After header and surfaced as user-friendly messages. API key 401/403 errors shown distinctly.
